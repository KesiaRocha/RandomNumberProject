How to Play: Random Number Guessing Game

Welcome to the tutorial for the Random Number Guessing Game! In this game, you'll try to guess a randomly generated number until you get it right.

Here's how it works:

  1. We'll generate a random number between a specified range (for example, 1 to 100).
  2. Your task is to guess the number.
  3. After each guess, we'll tell you if the number is too high, too low, or correct.
  4. Keep guessing until you guess the correct number!

Ready to play? Click the "PLAY" button to start the game!
